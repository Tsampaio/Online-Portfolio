Google Maps API Project

  • KnockoutJS
  • Ajax
  • Jquery
  • Google Maps API

This is a single-page application that I have completed for the Udacity Front End Developer Nanodegree. This project was finished using the Javascript framework Knockout.js and a third-party API Google Maps.

The page is loaded with some markers showcasing popular cocktail bars in Manchester, where by clicking on them we can check their location, streetview image, and website information.

The page is responsive but it is not intended for mobile viewing on small devices. If you are viewing on a mobile phone, landscape is recommended as the width of the search list is fixed.
